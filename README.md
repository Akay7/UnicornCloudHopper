# 🦄 Unicorn Cloud Hopper

A fun HTML5 browser game where you guide a unicorn flying between clouds. Don't fall!

**[Play it live →](https://akay7.github.io/UnicornCloudHopper/)**

## How to Play

- **Click**, **Tap**, or press **Space** to make the unicorn flap upward
- Land on clouds to stay alive
- Miss a cloud and fall off screen = Game Over
- Each cloud you pass scores a point ☁️

## Features

- 🦄 Unicorn with a rainbow trail
- ☁️ Scrolling clouds at varying heights
- 🌞 Animated spinning sun in the top-right corner
- ✨ Sparkle particle effects
- Score counter with best score tracking
- Mobile-friendly (tap to play)

## Running Locally

No build step needed — just open `index.html` in any modern browser.

```bash
git clone https://github.com/Akay7/UnicornCloudHopper.git
cd UnicornCloudHopper
open index.html   # macOS
# or: xdg-open index.html  (Linux)
# or: start index.html     (Windows)
```
